## Getting Started

First Clone git,
Second install npm packages by `npm i`. Then get build version:
```bash
npm run build
# or
yarn build
```
At the end run server by using :`yarn start`
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
